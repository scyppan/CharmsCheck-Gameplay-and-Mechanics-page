Backgrounds

There are four backgrounds that affect the number of traits versus initial skill "buys" you have during character creation.

A "buy" is a point one can spend when making a character or when the character is matriculating. Initial 'buys' are the buys one gets when making a character.

  • Pureblood: +0 Traits; +3 initial skill buys
  • Wizard-Raised Halfblood: +1 Traits; +2 initial skill buys
  • Muggle-Raised Halfblood: +2 Traits; +1 initial skill buys; +10 to Muggles skill
  • Muggleborn: +3 Traits; +0 initial skill buys; +10 to Muggles skill
